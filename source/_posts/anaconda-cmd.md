Anaconda指的是一个开源的Python发行版本，其包含了conda、Python等180多个科学包及其依赖项。Anaconda 本身支持虚拟环境，不需要安装pyenv.

<!-- more -->

## 虚拟环境

```bash
# create
conda create --name ml

# list env
conda info --envs

# active env
conda activate ml

# deactive
source deactivate
```
